GACKbot - Home Bot
******************

Usage
=====

Run the bot::

    export SLACKBOT_API_TOKEN='xoxb-XXXX-XXXX'
    gackbot


Installation
============

You'll need to create a Slack Bot and create a Slack Bot API Token, then::

    pip install gackbot
